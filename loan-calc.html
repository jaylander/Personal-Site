Loan Calculator

A single page website that the user enters loan amount, interest rate, and loan length in years. Calculates the monthly payments along with total interest, total loan cost, and an amortization schedule table. Applied Bootstrap elements to the UI and make it responsive for mobile browsers. Used a CSS library to animate the table when user calculates or resets the loan calculator.
